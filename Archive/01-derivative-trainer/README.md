# Derivative Trainer with GeoGebra

This project is an interactive tool designed to master the rules of differentiation. It combines a **random problem generator** with **real-time graphical feedback**, allowing users to practice calculating derivatives while visualizing the relationship between a function and its slope.

## ðŸ“– Overview

Mastering derivatives requires moving beyond memorization toward active practice. This applet provides a structured environment to train your skills using two main windows:

1.  **Control & Training Panel (Left):** This is where you interact with the algebra. You can input custom functions, generate random challenges based on specific rules, and verify your solutions.
2.  **Visualization Deck (Right):** This window provides the geometric context. It allows you to toggle the graphs of the function and its derivative to see how symbolic changes affect the geometry of the curve.

---

## ðŸ›  Key Features

* **Customizable Random Generator:** Generate endless problems by choosing the types of functions you want to practice (Polynomials, Trig, Exp, Log, etc.).
* **Difficulty Scaling:** Adjust the difficulty level to move from basic functions to more complex expressions involving products, quotients, compositions, etc.
* **Instant Verification:**
    * **Check Box:** Type your answer and use the **Check Answer** button to see if it is correct.
    * **Show Answer:** If you are stuck, you can reveal the correct derivative to compare with your work.
* **Visual Intuition:** Use the checkboxes to show the graph of $f(x)$ and $f'(x)$. This helps you visualize how the "zeros" of the derivative correspond to the "peaks and valleys" of the original function.
**Note:** For longer expressions it might be needed to move the **derivative solution** textbox to see the full answer, as it might not fit in the window.

## ðŸ”¬ Mathematical Context

The derivative $f'(x)$ represents the **instantaneous rate of change** of a function. Being able to find the correct derivative of a function is fundamental for physics, mathematics and many other disciplines, but getting the hang of it can be challenging at first and requires a lot of practice. With this applet you can generate as many practice problems as you want, whether you are a teacher or a student.

The function generator combines the chosen function types by randomly multiplying, adding, subtracting, dividing, composing or exponentiating them to generate a good practice problem.
